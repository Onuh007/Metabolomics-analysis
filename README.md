# Metabolomics Analysis Pipeline

## Overview
A complete R-based metabolomics analysis pipeline for comparing metabolite profiles between Control and Disease groups.

## Methods
- Missing value imputation using half-minimum method
- Log2 transformation and autoscaling
- Univariate statistical testing with Benjamini-Hochberg correction
- Principal Component Analysis (PCA)
- PLS-DA with VIP scoring
- Publication-ready visualisations

## Requirements
- R version 4.5
- Packages: tidyverse, FactoMineR, factoextra, mixOmics, corrplot

## Usage
Run `module8_pipeline.R` in RStudio with the provided dataset.
